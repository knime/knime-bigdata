### Content
This directory contains the source files for the Secured Cluster Connection Guide (Server). The code is organized as follows:

* _index.adoc_: The master document
* _server_*.adoc_: The document sections
* _img/*_: Image files for documentation

###Editing and viewing the documentation
To edit and view the documentation download and install the AsciidocFX Editor (https://asciidocfx.com/).

### Prerequisites for compiling the documentation

[http://asciidoctor.org/docs/install-toolchain/](Follow instructions here...)
sudo apt-get install asciidoctor

### Compiling the documentation

## Linux ##

cd into this directory.
asciidoctor index.adoc
