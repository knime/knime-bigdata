The Databricks JDBC Driver is a Type 4 JDBC driver that provides database connectivity through the 
standard JDBC application program interfaces (APIs) available in the Java Platform, Enterprise Editions. The Driver 
downloads are available to all Databricks users at no additional charge, and provide access to Databricks
from any Java application, application server, or Java-enabled applet.
